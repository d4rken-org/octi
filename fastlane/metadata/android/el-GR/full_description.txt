Το <i>Octi</i> συγχρονίζει πληροφορίες μεταξύ πολλαπλών συσκευών Android.

Διαθέσιμες ενότητες πληροφοριών:

* Λεπτομέρειες συσκευής & λειτουργικού συστήματος Android
* Κατάσταση μπαταρίας
* Συνδεσιμότητα WiFi
* Εγκατεστημένες εφαρμογές
* Κοινή χρήση προχείρου
* και περισσότερα... αν έχετε μια καλή ιδέα, ενημερώστε με!

Μπορείτε να επιλέξετε διαφορετικούς τρόπους για να συγχρονίζετε τις πληροφορίες:

* Το Google Drive Σας
* Ένας δωρεάν server που φιλοξενείται από εμένα
* Φιλοξενείστε τον δικό σας server
* Περισσότερα σύντομα...

Οι επιλογές συγχρονισμού μπορούν να αναμιχθούν.

Το Octi είναι ανοιχτού κώδικα, δεν έχει διαφημίσεις και δεν σας παρακολουθεί.
Μπορείτε να αγοράσετε μια αγορά εντός εφαρμογής για να πάρετε μερικές επιπλέον δυνατότητες και να υποστηρίξετε την ανάπτυξη της εφαρμογής.
